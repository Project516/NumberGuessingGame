About

Number Guessing Game

A simple number guessing game where you try to guess a randomly generated number between 1 and 100.

How to Play:

  • Enter a number between 1 and 100
  • Submit your guess
  • Follow the hints (too high/too low)
  • Try to guess the number in as few attempts as possible
  • Save your score and compete for the top spot!

Features:

  • Modern, responsive design
  • High score tracking with localStorage
  • Visual feedback for guesses
  • Mobile-friendly interface
